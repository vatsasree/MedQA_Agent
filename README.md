# MedQA Agent (Medical Clinical Insight Engine)

An autonomous AI Medical Agent powered by LangGraph, Streamlit, and a locally hosted `Qwen3.5-122B-A10B` large language model. 

The MedQA Agent is designed to intelligently search through medical handbooks for clinical guidelines and query patient reports to provide highly accurate, context-aware medical insights.

---

## 🚀 Quick Start

The entire stack (vLLM inference server and Streamlit frontend) is managed seamlessly via a single bash script that handles GPU environment configuration, NCCL networking (to prevent multi-GPU segfaults), and background `tmux` sessions.

### 1. Launch the Stack
Simply run the startup script:
```bash
./start_all.sh
```

### 2. Monitor the Services
The script launches the services in the background. You can drop in and monitor their logs at any time using `tmux`:

* **Monitor vLLM (Model Loading & Inference):**
  ```bash
  tmux a -t medqa_vllm
  ```
* **Monitor Streamlit (Frontend Logs):**
  ```bash
  tmux a -t medqa_app
  ```
*(To safely exit a tmux session without killing the process, press `Ctrl+B`, then `D`)*

### 3. Shutting Down
To safely shut down the background processes and free up your GPUs:
```bash
tmux kill-session -t medqa_vllm
tmux kill-session -t medqa_app
```

## 🛠️ Architecture

* **Frontend:** Streamlit (`app.py`) providing a chat interface.
* **Agent Engine:** LangChain/LangGraph orchestration (`src/agent.py`) capable of reasoning and utilizing tools.
* **LLM Backend:** `vLLM` running on port `8000`. The server is explicitly configured with `--tool-call-parser qwen3_xml` to correctly intercept the native XML tool calls generated by Qwen3.5.
* **Vector Store:** Local ChromaDB/FAISS (depending on `src/ingest.py` implementation) for RAG capabilities.

## 💡 Sample Questions

Once the agent is online, you can ask it a variety of questions. The agent will autonomously decide whether to search the medical handbook or query specific patient reports.

**General Clinical Guidelines (Handbook)**
* *"What are the standard first-line treatments for acute hypertension?"*
* *"How should I treat severe stomach pain that occurs exclusively at night during sleep?"*
* *"What are the clinical guidelines and potential side effects for treating migraines with cryotherapy?"*
* *"What are the primary contraindications for prescribing beta-blockers to asthma patients?"*

**Patient-Specific Queries (Reports)**
*(Assuming patient PDFs have been ingested via `src/ingest.py`)*
* *"Can you pull John Doe's latest lab results and check if his lipid levels are elevated?"*
* *"Based on Jane Smith's historical reports, is there a record of allergic reactions to penicillin?"*
* *"Summarize the post-operation recovery vitals for patient ID #10425."*
